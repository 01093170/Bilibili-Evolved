# Bilibili-Evolved
增强哔哩哔哩Web端体验的油猴脚本.

作为[原脚本](bilibili-touch.js)的工程化形式,工程化目前未完成,完成之前可以先使用原脚本.

# 主要功能
## 新样式
改变顶栏的样式,适用于主站和播放界面.暂不支持直播,相簿.
